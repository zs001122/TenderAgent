## 技术决策记录

| 决策 | 选择 | 原因 |
|------|------|------|
| 爬虫框架 | 自研 + Playwright | 灵活可控，支持动态页面 |
| 去重方式 | URL去重 | 简单有效 |
| 增量抓取 | 基于发布日期 | 避免重复抓取 |
| 调度方式 | 单进程顺序执行 | 简单可靠 |
