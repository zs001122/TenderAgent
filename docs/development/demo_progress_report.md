# 招标商机挖掘系统 Demo 实施进度报告

## 1. 概览
本项目旨在构建一个智能化的招标信息挖掘与分析系统。目前已完成第一阶段的 Demo 开发，实现了从特定渠道抓取数据到利用大模型进行商机分析的完整流程。

## 2. 已实现功能模块

### 2.1 数据抓取层 (Data Collection)
- **目标源**: 中国电信阳光采购网 (https://caigou.chinatelecom.com.cn)
- **技术栈**: Python + Playwright (Async) + Pandas
- **功能**:
    - 自动遍历公告列表页
    - 抓取公告基础信息（标题、日期、省份、公告类型等）
    - 抓取详情页内容（包括正文文本、附件列表）
    - 自动处理分页与反爬策略（基础的延时等待）
    - 数据持久化存储为 CSV 格式 (`中国电信采购公告.csv`)
- **脚本文件**: `chinatelecom_playwright_scraper.py`

### 2.2 智能分析层 (AI Analysis)
- **输入**: 抓取的 CSV 原始数据
- **技术栈**: Python + OpenAI SDK (兼容接口) + Regex
- **功能**:
    - **基础提取**: 
        - 标题清洗（处理 Excel Hyperlink 公式）
        - 预算金额提取（支持"万元"、"亿元"、"元"等多种格式）
        - 截止时间提取
        - 行业标签提取（基于关键词库匹配：大数据、AI、通信、工程等）
    - **AI 深度分析**:
        - 集成 LLM (Qwen3-32B / gpt-oss-120b) 进行非结构化文本分析
        - 自动评估：风险等级、竞争对手推测、技术难度、综合建议
        - 输出结构化 JSON 数据
    - **商机匹配**:
        - 基于预设的"公司画像"（如：主营大数据/AI、预算范围 50-2000万）
        - 多维度打分算法（业务匹配度 40% + 预算匹配度 30% + 资质匹配度 30%）
        - 生成推荐等级（强烈推荐/一般推荐/不推荐）
- **脚本文件**: `tender_analysis_demo.py`

## 3. 运行结果示例

### 3.1 成功案例
- **项目**: 2026年阳江分公司资本性支出项目监理服务框架采购
    - **评分**: 70分 (一般推荐)
    - **AI 建议**: 建议具备通信行业多类型项目监理经验的企业积极投标，重点关注技术综合能力与本地服务能力。
- **项目**: 云南百事通信息分公司2026年产品运营支撑外包服务
    - **评分**: 70分 (一般推荐)
    - **AI 建议**: 建议具备综合运营支撑能力和本地化服务网络的企业重点关注并参与投标。

### 3.2 负样本识别
- **项目**: 阜阳分公司2026年第一批小微ICT设备采购
    - **评分**: 30分 (不推荐)
    - **原因**: 识别为硬件设备采购，与当前公司主营业务（软件/服务）不符。
